# Campus Timetable Scheduler - 10-Phase Development Plan

## Project Overview
A comprehensive timetable scheduling system for colleges with:
- ✅ Common Timetable (generated by College Admin)
- ✅ Approval Workflow (Lab Incharge → Common Subject Heads → Lock)
- ✅ Department Timetable (HOD assigns workload, generate timetable)
- ✅ Suggestion-based + Manual Override
- ✅ PostgreSQL + Django + React
- ✅ Custom Admin Panel

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/auth/login/ | Login |
| POST | /api/auth/logout/ | Logout |
| GET | /api/users/me/ | Current user |

### Core Management
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET/POST | /api/colleges/ | Colleges |
| GET/POST | /api/departments/ | Departments |
| GET/POST | /api/labs/ | Labs |
| GET/POST | /api/staff/ | Staff members |

### Subjects & TimeSlots
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET/POST | /api/subjects/ | Subjects |
| POST | /api/timeslots/generate/ | Generate timeslots |

### Common Timetable
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/common-timetable/generate/ | Generate common TT |
| POST | /api/common-timetable/save/ | Save suggestion |
| POST | /api/common-timetable/submit/ | Submit for approval |

### Approvals
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/approvals/pending/ | Pending approvals |
| PATCH | /api/approvals/{id}/approve/ | Approve |
| PATCH | /api/approvals/{id}/reject/ | Reject |

### Workload
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET/POST | /api/workload-assignments/ | Assignments |
| POST | /api/workload-config/ | Configure limits |

### Department Timetable
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/department-timetable/generate/ | Generate |

| Method | Endpoint | Description |
|--------|----------|-------------|

| GET | /api/timetable/merged/conflicts/ | Check conflicts |
| GET | /api/timetable/export/ | Export (CSV/Excel/PDF) |

## 🎯 USER FLOW SUMMARY

### College Admin Flow
1. Login → Admin Dashboard
2. Configure subjects (common + core)
3. Manage labs
4. Generate Common Timetable
5. Submit for Approval
6. Wait for approvals

### Lab Incharge Flow
1. Login → Approval Dashboard
2. View pending common timetable
3. Approve/Reject with comments

### Common Subject Head Flow
1. Login → Approval Dashboard
2. View pending common timetable
3. Approve/Reject with comments

### HOD Flow
1. Login → Department Dashboard
2. View staff workload status
3. Assign subjects to staff (workload allocation)
4. Generate Department Timetable
5. Review suggestions
6. Manual override if needed
7. Publish Department Timetable

### Staff Flow
1. Login → View personal timetable
2. See assigned classes
3. See workload status

### Student Flow
1. Login → View Timetable
2. See merged (common + department) timetable
3. No free hours shown
